## Abstract (EN)

This paper proposes an independently originated theoretical claim: **an AI Token Relay Station is, in its functional substance, a Token Bank.** The paper argues that a Token Relay Station fully replicates the four core functions of commercial banking — aggregation, maturity transformation, risk pooling, and credit creation — and that from this foundation a complete portfolio of financial products can be derived. The paper further defines **the Token Bank as a new form of Virtual Bank** whose business scope intrinsically includes digital currency settlement and collateralized token lending (Compute Lending). Through a structural comparison with the GitHub aggregation model, the paper demonstrates that the Token Bank's flywheel mechanics, gross-margin structure, and competitive moat differ fundamentally from those of traditional knowledge-aggregation platforms.

## 摘要（中）

本文提出一个独立原创的理论命题：**AI Token 中转站本质上是一种 Token 银行**。本文论证 Token 中转站完整复刻了商业银行的四大核心功能——聚合、期限转换、风险池化、信用创造——并由此派生出涵盖利差套利、智能路由套利、Token 期货、Token 信用透支、Token 收益凭证、Token SLA 保险等六大金融产品。本文进一步定义"Token 银行"为一种新型**虚拟银行**形态，其业务范围天然包含数字货币交易与抵押化 Token 购买（Compute Lending）。通过与 GitHub 聚合模式的结构性对比，本文论证 Token 银行的飞轮机制、毛利结构与护城河属性与传统知识聚合平台具有根本性差异。
